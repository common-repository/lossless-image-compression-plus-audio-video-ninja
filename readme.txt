Lossless image compression now offers audio and video compression! Why sacrifice quality for Google PageSpeed, get both.

== Description ==

The only Audio and Video optimizer that comes with lossless image compression built right in. Take all of your Media and turn it into high speed content with our compression plugin. Using mediacru.sh API to compress it is sure to be one of the most stable and effective plug ins for your website.

== Installation ==

1. Upload the 'lossless-image-compression-plus-audio-video-ninja' plugin to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Done!

== Frequently Asked Questions ==

= Does the plugin replace existing images, video, and audio that I have uploaded ot my site? =

Absolutely! Only if the optimized version is smaller. The plugin should NEVER create a larger image.
